함수

함수도 변수처럼 하나의 데이터 타입입니다.
따라서 함수를 변수에 대입하거나 함수에 프로퍼티를 지정하는것도 가능합니다. 또한 함수내에 다른 함수도 중첩되어 정의돌 수도 있습니다.
이같은 점을 바탕으로 함수는 프로그램을 구성하는 주요 구성요소이며 함수를 사용하면 중복없이 유사한 동작을 하는 코드를 여러번 호출할 수 있습니다.
함수의 구성요소는 JS코드를 저장하고, 펑션이라는 키워드로 선언합니다. 그리고 출력문, 제어문등의 코드를 저장하고 반환합니다.


선언적 함수

문법

function 함수명(){
    JS코드
}
함수명();

선언적 함수는 함수를 선언할때 이름을 붙여주는 함수를 말한다. 호출할때에는 붙여준 이름으로 호출할수있다.

다음은 선언적 함수를 이용해 [배경색 바꾸기]버튼을 클릭할 때마다 배경색이 바뀌도록 만든 예제입니다.

<script>
    var color = ["white", "yellow", "aqua", "purple"];

    var i = 0;
    function changeColor(){
        i++;
        if (i >=color.length){
            i = 0;
        }                    // i의 값이 4 이상이면 i에 0을 대입하는 함수입니다.
    

        var bodyTag = document.getElementByld("theBody");
        bodyTag.style.backgroundColor = color[i];
    }
</script>
</head>
<body id="theBody">
    <button onclick="changeColor();">배경색 바꾸기</button>    // 버튼 클릭시 changecolor() 함수를 호출합니다.
</body>

익명 함수

문법

const 함수명 = function(){
    JS코드               
}
함수명();

익명함수는 이름에서 알수있듯이 이름이 없는 함수다. 그래서 사용하기 위해서는 변수에 넣어서 사용해야 한다.


선언적 함수와 익명함수의 차이점

보통 웹브라우저가 script 태그를 읽을때 위에서부터 아래로 차례대로 읽는다.
그러나 선언적 함수의 경우 그것을 무시하고 최우선으로 읽기때문에 함수호출문을 함수 위쪽에 적어놓아도 정상적으로 작동한다.
하지만 익명함수의 경우 그에 해당하지않기 때문에 함수가 함수호출보다 아랫쪽에 있을시 오류가 나게된다.


매개변수 함수

문법

{
    function func(str){
        document.write(str);
    }
    func("함수가 실행되었습니다.")
}

선언적 함수의 경우 함수를 실행할때 단순히 함수명으로 호출해서 실행했습니다.
하지만 매개변수가 있는 함수는 호출할때 전달하고자 하는 값을 입력해서 호출할수 있습니다. 이렇게 전달된 값은 매개변수가 받아 함수에서 사용할수 있습니다.
