자바스크립트 선언문

선언문은 자바스크립트 코드를 작성할 영역을 선언하는 것이라고 이해하면 됩니다.
<script>라는 태그로 선언문이 시작된 곳부터 <script>라는 태그로 종료된 곳까지를 스크립트 영역이라고 부릅니다.

선언문은 <head>태그 영역 또는 <body>태그 영역에 선언하면 됩니다.
우리나라 개발자의 대부분은 <head>태그 영역에 선언하고 있습니다. 

<!DOCTYPE html>
<html lang="ko">
    <head>
        <meta charset="UTF-8">
        <title>Document</title>
        <script>
        document.write("환영합니다")
        </script>
    </head>
<body>
</body>
</html>

자바스크립트 주석처리

자바스크립트 선언문 안에서 코드가 아닌 설명글을 넣고 싶을 때는 주석 처리를 할수있다.
이때 한줄 주석일 경우에는 // 한줄 설명글로 작성할수있고, 여러줄 주석일경우는 /* 주석 */ 로 작성할수있다.

// 한줄 설명글
/*
여러줄
설명할때는
이렇게
*/

코드 입력시 주의사항

다음은 자바스크립트를 작성할 때 주의해야 할 몇 가지 사항입니다.

대 / 소문자를 구분하여 작성한다.

날짜 객체 생성: New date(); ( X )
날짜 객체 생성: new Date(); ( O )

코드 작성후에는 세미콜론(;)을 붙여줘야하며, 세미콜론을 쓰지않으면 한줄에 2개의 코드를 작성할경우 오류가 발생한다.


document.write("hi") document.write("bye")    ( X )

document.write("hi"); document.write("bye");  ( O )

코드 작성시에는 위처럼 한줄에 여러개 작성보다는 한줄만 작성하는것이 가독성이 좋다.

p>

문자형 데이터를 작성할 때는 큰따옴표(" ")와 작은따옴표('')의 겹침 오류를 주의해야 합니다.

큰따옴표 겹침 오류
documente.write("책에 "자바스크립트는 대소문자를 구분해야합니다"라고 나와 있다.");

잘된 예
documente.write('책에 "자바스크립트는 대소문자를 구분해야합니다"라고 나와 있다.'');
documente.write("책에\"자바스크립트는 대소문자를 구분해야합니다\"라고 나와 있다.");
