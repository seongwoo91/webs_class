for 문

기본적으로 for문은 초깃값, 조건식, 증감식으로 구성되어 있습니다.

1. 초기화 구문인 초깃값이 존재한다면 초깃값이 실행됩니다 보통 1이나 반복문 카운터로 초기 설정이 됩니다.
그러나 복잡한 구문으로 표현 될 때도 있습니다 또한 변수로 선언 되기도 합니다.

2. 조건문은 조건을 검사합니다 만약 조건문이 참이라면, 그 반복문은 실행됩니다.
만약 조건문이 거짓이라면, 그 for문은 종결됩니다 만약 그 조건문이 생략된다면, 그 조건문은 참으로 추정됩니다.

3. 문장을 실행합니다. 많은 문장을 실행할 경우엔, { } 를 써서 문장들을 묶어줍니다.
갱신 구문인 증감식이 존재한다면 실행되고 2번째 단계로 돌아갑니다.

아래는 예시입니다. 

let array = [100, 200 ,300];
for (let i=0; i < array.length; i++){
    document.write(arr[i]);
}    
/**
100
200
300
*/

for Each 문

forEach 반복문은 오직 Array 객체에서만 사용가능한 메서드입니다.
배열의 요소들을 반복하여 작업을 수행할 수 있습니다.
forEach구문의 인자로 callback 함수를 등록할 수 있고, 배열의 각 요소들이 반복될 때 이 callback 함수가 호출됩니다.
callback 함수에서 배열요소의 인덱스와 값에 접근할수 있습니다. 

const num = [100, 200, 300, 400, 500];
//forEach(element, index, array)
num.forEach(function(element, index, array){
    document.write(element);
    document.index(element);
    document.array(element);
})
