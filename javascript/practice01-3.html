테이블 출력하기

1.1 for문을 이용해서 테이블을 출력하시오

 let table = "<table>";

for (let i = 1; i <= 5; i++) {
    table += "<tr>";
    for (let j = 1; j <= 5; j++) {
        table += "<td>1</td>";
    }
    table += "</tr>";
}
table += "</table>";

document.write(table);

1.2 for문을 이용해서 테이블을 출력하시오

1부터 25까지 출력하기

{
    let num = 1;
    let table = "<table>";
        
    for (let i = 1; i <= 5; i++) {
        table += "<tr>";
        for (let j = 1; j <= 5; j++) {
            table += "<td>" + num + "</td>";
            num++;
        }
        table += "</tr>";
    }
    table += "</table>";   
}
