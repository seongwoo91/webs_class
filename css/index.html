CSS 레퍼런스 북

CSS 레퍼런스 속성과 설명을 나타내는 표입니다.
속성 설명
align-content align-content 속성은 콘텐츠 사이와 주위 빈 공간을 플렉스박스의 교차축, 그리드의 블록축을 따라 배치하는 방식을 결정합니다.
align-items align-items 속성은 모든 직계자식의 align-self 값을 그룹으로 설정합니다 플렉스박스에서는 Cross Axis의 항목 정렬을 제어하며, 그리드 레이아웃에서는 그리드 영역내의 블록 축의 항목 정렬을 제어합니다.
align-self align-self 속성은 그리드 또는 플렉스 항목의 align-items 값을 재정의합니다 그리드에서는 그리드 영역 안에 항목을 정렬합니다 플렉스박스에서는 항목을 교차 축에 정렬합니다.
all all 단축 속성은 요소의 unicode-bidi(en-US), direction(en-US), CSS 사용자 지정 속성을 제외한 모든 속성을 초기화 합니다. 초깃값, 상속값, 아니면 다른 스타일시트 출처의 값으로 설정할 수 있습니다.
animation animation 속성은 다수의 스타일을 전환하는 애니메이션을 적용합니다.
animation-delay animation-delay 속성은 애니메이션이 시작할 시점을 지정합니다 시작 즉시, 잠시 후에, 또는 애니메이션이 일부 진행한 시점부터 시작할 수 있습니다.
animation-direction animation-direction 속성은 애니메이션이 앞으로, 뒤로 또는 앞뒤로 번갈아 재생되어야하는지 여부를 지정합니다.
animation-duration animation-duration 속성은 애니메이션이 한 사이클을 완료하는데 걸리는 시간을 지정합니다.
animation-fill-mode animation-fill-mode 속성은 CSS 애니메이션이 실행 전과 후에 대상에 스타일을 적용하는 방법을 지정합니다.
animation-iteration-count animation-iteration-count 속성은 애니메이션 시퀀스가 중지되기 전에 재생되어야 하는 횟수를 설정합니다.
animation-name animation-name 속성은 요소에 적용할 애니메이션을 설명하는 규칙에서 하나 이상의 키프레임 이름을 지정합니다.
animation-play-state animation-play-state 속성은 애니메이션이 실행 중인지 일시 중지 되었는지 설정합니다.
animation-timing-function animation-timing-function 속성은 각 주기의 지속 시간 동안 애니메이션이 진행되는 방식을 설정합니다.
backface-visibility backface-visibility 속성은 요소의 뒷면이 사용자를 향할 때 보여야 하는지 지정합니다.
background background 속성은 색상, 이미지, 원점, 크기, 반복 등 여러 배경 스타일을 한 번에 지정합니다.
background-attachment background-attachment 속성은 배경 이미지를 뷰포트 내에서 고정할지, 아니면 자신의 컨테이닝 블록과 함께 스크롤할지 지정합니다.
background-blend-mode background-blend-mode 속성은 배경 이미지가 서로, 그리고 요소의 배경색과 어떻게 혼합되어야 하는지를 설정합니다.
background-clip background-clip 속성은 요소의 배경이 테두리, 안쪽 여백, 콘텐츠 상자 중 어디까지 차지할지 지정합니다.
background-color background-color 속성은 요소의 배경 색을 지정합니다.
background-image background-image 속성은 요소의 배경 이미지를 한 개나 여러 개 지정합니다.
background-origin background-origin 속성은 배경의 원점을 테두리 시작점, 테두리 내부, 안쪽 여백 내부 중 하나로 지정합니다.
background-position background-position 속성은 각 배경 이미지의 초기 위치를 설정합니다. 위치는 background-origin에 의해 설정된 위치레이어에 상대적입니다.
background-repeat background-repeat 속성은 배경 이미지의 반복 방법을 지정합니다. 가로축 및 세로축을 따라 반복할 수 있고, 아예 반복하지 않을 수도 있습니다.
background-size background-size 속성은 요소 배경 이미지의 크기를 설정합니다 그대로 두거나, 늘리고 줄이거나, 공간에 맞출 수 있습니다.
border border 속성은 요소의 테두리를 설정합니다. border-width, border-style, border-color의 값을 설정합니다.
border-bottom border-bottom 속성은 요소의 아래쪽 테두리를 설정합니다 border-bottom-width, border-bottom-style, border-bottom-color의 값을 지정합니다.
border-bottom-color border-bottom-color 속성은 요소의 아래쪽 테두리 색상을 지정합니다 border-color 또는 border-bottom 단축 속성으로도 지정할 수 있습니다.
border-bottom-left-radius border-bottom-left-radius 속성은 모서리의 곡률을 정의하는 타원의 반지름(또는 반장축 및 반단축의 반지름)을 지정하여 요소의 왼쪽 아래 모서리를 반올림합니다.
border-bottom-right-radius border-bottom-right-radius 속성은 모서리의 곡률을 정의하는 타원의 반지름(또는 반장축 및 반단축의 반지름)을 지정하여 요소의 오른쪽 아래 모서리를 반올림합니다.
border-bottom-style border-bottom-style 속성은 요소 테두리의 아래쪽 스타일을 지정합니다.
border-bottom-width border-bottom-width 속성은 요소의 아래 테두리 너비를 지정합니다.
border-collapse border-collapse 속성은 표 테두리(border)가 분리(separated) 또는 상쇄(collapsed)될 지를 결정합니다. 분리 모델에서는, 인정합 셀은 각각 자신의 고유(distinct) 테두리가 있습니다. 상쇄 모델에서는, 인접한 표 셀은 테두리를 공유합니다.
border-color border-color 속성은 모든 면의 테두리 색상을 결정합니다.
border-image border-image 속성은 요소의 주위에 이미지를 그립니다. 일반 테두리를 대체합니다.
border-image-outset border-image-outset 속성은 요소의 테두리 상자와 테두리 이미지의 거리를 설정합니다.
border-image-repeat border-image-repeat 속성은 원본 이미지의 모서리 영역을 요소의 테두리 이미지 크기에 맞춰 조절할 때 사용할 방법을 지정합니다.
border-image-slice border-image-slice 속성은 border-image-source로 설정한 이미지를 여러 개의 영역으로 나눕니다. 이렇게 나눠진 영역이 요소의 테두리 이미지를 이룹니다.
