자바스크립트 선언문

선언문은 자바스크립트 코드를 작성할 영역을 선언하는 것이라고 이해하면 됩니다.
<script>라는 태그로 선언문이 시작된 곳부터 <script>라는 태그로 종료된 곳까지를 스크립트 영역이라고 부릅니다.

선언문은 <head>태그 영역 또는 <body>태그 영역에 선언하면 됩니다.
우리나라 개발자의 대부분은 <head>태그 영역에 선언하고 있습니다. 

<!DOCTYPE html>
<html lang="ko">
    <head>
        <meta charset="UTF-8">
        <title>Document</title>
        <script>
        document.write("환영합니다")
        </script>
    </head>
<body>
</body>
</html>

자바스크립트 주석 처리

자바스크립트 선언문 안에서 코드가 아닌 설명글을 넣고 싶을 때는 주석 처리를 하면 됩니다.
이때 한줄 주석일 경우에는 // 한줄 설명글로 작성할수있고, 여러줄 주석일경우는 /* 주석 */ 로 작성할수있다. 

// 한줄 설명글

/* 

여러줄

설명할때는

이렇게

*/

주석이 필요한경우나, 메모하는경우 그리고 여러사람이 작업할경우 코드에 대해 설명하는 용도로 다는등 활용법이 많다

내부 스크립트 외부로 분리하기

HTML 내부에 작성된 자바스크립트는 마지막에는 외부로 분리하는 작업을 해야한다.
그렇게 하면 자바스크립트 소스 찾기도 쉬울 뿐만 아니라 누군가가 소스를 손상시킬 염려도 없어진다.
즉, 외부로 분리하는 주된 목적은 프로젝트 관리를 원할하게 하기 위함이며 다음으로는 외부 자바스크립트 연동 코드로 파일을 연결하는 기본형이다. 


기본형 : <script src="JS 파일경로"></script>

내부 스크립트를 외부로 분리하려면 따로 파일을 만들어서 저장해야 한다.
즉, HTML 파일과 JS(Javascript)파일을 분리하여 HTML 내에서 JS(Javascript)파일을 연동하면 된다.


예제

01:<!DOCTYPE html>
02:<html lang="ko">
03:<head>
04:   <meta charset="UTF-8">
05:   <title>외부 자바스크립트 연동</title>
06:   <scrips src="자바스크립트파일.JS"></scrips>
07:</head>
08:<body>
09:</body>
10:</html>


위의 예제와 같이 따로 작업한 JS(Javascript)파일을 HTML에 연동시킬수있다.
