CSS란?

CSS(Cascading Style Sheets)는 HTML 같은 문서의 스타일을 꾸밀 때 사용하는 스타일 시트 언어입니다.
글꼴, 배경색, 너비와높이, 위치등을 조정하거나 웹브라우저의 스크린 크기, 장치에 따라서 화면을 다르게 표시하도록 지정할수도 있습니다.

css 주석 표시방법

/* CSS 주석 */

CSS 선언 방법

내부 스타일

HTML 파일 안에 <style> 태그로 스타일을 작성하는 방식입니다.

<style>
    .doc {
        font-size: 18px;
        line-height: 1.8;
        margin-bottom: 200px;
    }
</style>

외부 스타일

link 태그를 이용하여 외부 CSS 파일을 연결하는 방식입니다.

<link rel="stylesheet" href="assets/css/style.css">

인라인 스타일

요소의 style 속성에 직접 스타일을 작성하는 방식입니다.
최우선 순위이기 때문에 외부스타일과 혼합하여 사용하는경우 추후 수정에 어려움이 따를수있는 방식입니다.

<div style="color:black; margin:20px;">
    ...
</div>
