SEO(검색엔진 최적화)란?

SEO는 웹사이트가 유기적인 검색 방식을 통해 검색엔진에서 상위 에 노출될 수 있도록 최적화하는 과정을 말합니다.
검색엔진은 사용자들에게 최고의 정보를 제공하길 원하기 때문에 SEO를 통해 내가 만든 콘텐츠가
특정 검색어에 맞춰 웹 상의 주요정보로 인식하도록 하는게 목표입니다.
SEO가 이루어지는 과정은 해당 페이지가 어떤 콘텐츠를 가지고있는가, 무엇에 대한 내용을 담고있는가 판단하기 위해 웹 크롤러를 사용합니다.
웹 크롤러는 코드를 스캔하여 웹에 표시되는 텍스트,이미지,동영상 등의 가능한 모든 정보를 수집하며,
해당 내용이 검색자에게 유용하다고 판단되면 색인에 추가하여 노출될수 있도록 합니다. SEO의 목적으로는
광고에 돈을 투자하지않고도 자신의 컨텐츠를 검색상위에 노출시키는 마케팅 전략입니다.
