ID 와 CLASS의 차이점

id와 class 는 특정 요소를 대상으로 스타일을 적용하기 위해 사용하는 것이며 둘의 차이점을 알아보겠습니다.


  • CSS의 정의방법이 ID는 # 으로, CLASS는 . 으로 정의합니다.
  • 하나의 요소에 ID는 하나만 적용가능하고, CLASS는 여러개 적용이 가능하다.
  • ID의 속성이 CLASS보다 우선순위가 높습니다.
