CSS 색상

CSS 색상은 색상, 색상 유형, 색상 혼합, 투명도 및 이러한 색상 효과를 HTML에 적용하는 방법을 다루는 CSS 모듈입니다.
css 색상은 다음과 같은 방법으로 사용할 수 있다.


  • 키워드 사용 (red, transparent 등)
  • rgb(), rgba()의 함수형 표기법
  • # + 16진수 표기법
  • HSL 실린더형 좌표계 사용 (hsl(), hsla() 의 함숳형 표기법)
