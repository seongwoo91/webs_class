기본 선택자

속성 예시 설명
태그 선택자 p {color: #fff} p 태그를 선택하여 글씨 색을 하얀색으로 변경합니다.
클래스 선택자 .class {color: #fff} class 클래스를 선택하여 글씨 색을 하얀색으로 변경합니다.
아이디 선택자 #id {color: #fff} id 아이디를 선택하여 글씨 색을 하얀색으로 변경합니다.
그룹 선택자 div, p {color: #fff} div태그와 p태그를 선택하여 글씨 색을 하얀색으로 변경합니다.
전체 선택자 * {color: #fff} 전체 태그를 선택하여 글씨 색을 하얀색으로 변경합니다.

계층 선택자

속성 예시 설명
하위 선택자
(Descendant Selector)		 div p {color: #fff} div 태그 하위의 p 태그를 선택하여 글씨색을 하얀색으로 변경합니다.
자식 선택자
(Child) Selector)		 div > p {color: #fff} div 태그의 첫번째 자식 p 태그를 선택하여 글씨색을 하얀색으로 변경합니다.
형제 선택자
(Sibling Selector)		 div ~ p {color: #fff} 같은부모를 가지며, div 뒤에 오는 P에 적용합니다.(인접과 달리 바로 뒤가 아니어도됨)
인접 형제 선택자
(Adjacent Sibling Selector)		 div + p {color: #fff} 같은 부모를 가지고있을시에 div 바로 뒤에 오는 p 태그의 글씨색을 하얀색으로 변경합니다.

속성 선택자

종류 예시 설명
[name] a[href] a 태그 속성 중에 href가 있으면 선택합니다.
[name="value"] a[href="#header"] a 태그 href 속성의 값이 '#header'이면 선택합니다.
[name~="value"] a[href~="web"] a 태그 href 속성의 값이 'web'을 포함한 요소를 선택합니다.
[name|="value"] a[href|="web"] a 태그 href 속성의 값이 'web'이거나 'web'으로 시작하는 요소를 선택합니다.
[name^="value"] a[href^="web"] a 태그 href 속성의 값이 'web'으로 시작하는 태그를 선택합니다.
[name*="value"] a[href*="web"] a 태그 href 속성의 값이 'web'이 포함되어 있는 태그를 선택합니다.
[name$="value"] a[href$="com"] a 태그 href 속성의 값이 'com'으로 끝나는 태그를 선택합니다.

가상 선택자

속성 예시 설명
::first-line p::first-line {color: #fff;} 요소의 첫번째 줄의 글씨색을 하얀색으로 변경합니다. 블록 요소에만 적용 가능합니다.
::first-letter p::first-letter {color: #fff;} 요소의 첫번재 글자의 글씨색을 하얀색으로 변경합니다. 블록 요소에만 적용 가능합니다.
::before p::before {content:'시작'} 선택한 요소 앞에 "시작" 을 삽입하며 before/after를 사용할때는 content를 사용합니다.
::after p::after {content:'끝'} 선택한 요소 뒤에 "끝" 을 삽입하며 before/after를 사용할때는 content를 사용합니다.
::selection p::selection {color:red} 선택한 요소를 마우스로 드래그 할시 색깔을 빨간색으로 바꿔줍니다.

가상 클래스

속성 예시 설명
:first-child div p:first-child {color: #fff;} div의 첫번째 자식 p요소의 글자색을 하얀색으로 변경합니다. 첫번째 자식이 p가 아닐경우 발동되지않습니다.
:last-child div p:last-child {color: #fff;} div의 마지막 자식 p요소의 글자색을 하얀색으로 변경합니다. 마지막 자식이 p가 아닐경우 발동되지않습니다.
:first-of-type div p:first-of-type {color: #fff;} div의 첫번째 자식 p태그에 자식의 글자색을 하얀색으로 변경합니다.
:last-of-type p:last-of-type {color: #fff;} div의 마지막 자식 p태그에 자식의 글자색을 하얀색으로 변경합니다.
:nth-child(n) p:nth-child(n) {color: #fff;} 부모의 p요소중 n번째 요소의 글자색을 하얀색으로 변경합니다.
:nth-last-child(2) p:nth-last-child(2) {color: #fff;} 부모의 p요소중 n번째 요소의 글자색을 하얀색으로 변경합니다. (n번째는 뒤에서부터 카운트합니다)
:nth-of-type(n) p:nth-of-type(n) {color: #fff;} 같은 유형의 p라는 요소중 n번째 요소의 글자색을 하얀색으로 변경합니다.
:nth-last-of-type(2) p:nth-last-of-type(2) {color: #fff;} 같은 유형의 p라는 요소중 n번째 요소의 글자색을 하얀색으로 변경합니다.
:only-child p:only-child {color: #fff;} 부모의 유일한 자식인 p요소의 글자색을 하얀색으로 변경합니다.
:only-of-type p:only-of-type {color: #fff;} 같은 유형의 유일한 형제인 P요소의 글자색을 하얀색으로 변경합니다.
:not li:not(class)) {color: #fff;} li요소중 class를 제외한 나머지 모든태그의 글자색을 하얀색으로 변경합니다.
:root :root {color: #fff;} --로 변수를 설정하고 일괄적으로 글자색을 하얀색으로 변경합니다.
:empty :empty {backgroud: #fff;} 자식이 없는 빈 요소의 배경색을 하얀색으로 변경합니다.

링크 가상 선택자

속성 예시 설명
:link a:link {color: #fff} 현재 문서에 연결된 외부문서를 표시하는 글씨의 색깔을 하얀색으로 변경합니다.
:visited a:visited {color: #fff} 방문했던 링크의 글씨색깔을 하얀색으로 변경합니다.
:hover a:hover {color: #fff} 링크의 글씨에 마우스를 올렸을때 글씨색깔을 하얀색으로 변경합니다.
:active a:active {color: #fff} 링크를 클릭했을때 글씨의 색깔을 하얀색으로 변경합니다.

폼 선택자

속성 예시 설명
:checked input:checked {color: #fff} input요소를 선택하였을때 대상의 색깔을 하얀색으로 변경합니다.
:focus input:focus {color: #fff} input 요소에 커서가 클릭하여 선택된 동안의 색깔을 하얀색으로 변경합니다.
:enabled input:enabled {color: #fff} input 요소중 활성화되어있는 영역의 색을 하얀색으로 변경합니다.
:disabled input:disabled {color: #fff} input 요소중 비활성화되어있는 영역의 색을 하얀색으로 변경합니다.
